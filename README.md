# 🚀 React Applications Portfolio

This repository showcases a collection of applications demonstrating **static**, **responsive**, and **dynamic** web development concepts.

---

## 📌 Overview

This project is built to demonstrate:

* Clean UI design
* Responsive layouts for multiple devices
* Dynamic data handling using React
* Component-based architecture

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3 / Flexbox / Grid
* Git & GitHub


## 🌟 Projects Included

### 1. Static App

* Simple UI 
* Focus on layout and styling
* No dynamic data

### 2. Responsive App

* Fully responsive design
* Works across mobile, tablet, and desktop
* Uses Flexbox/Grid and media queries

### 3. Dynamic App

* Interactive UI
* Handles user input and dynamic data
* Demonstrates real-world js functionality

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate to project folder
cd your-repo-name

# Install dependencies
npm install

# Start development server
npm start
```
